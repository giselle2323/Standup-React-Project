# Node JS ES6 Boilerplate

> A basic node js boiler plate with ES6 configurations, express and a recommended folder structure.
---
## Usage

- Clone repo
- Run `yarn install` to install dependencies
- Run `yarn start` to start server
---
## Technologies
- Node JS
- Express
